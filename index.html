Smart VIN Decoder

Developed a VIN decoder in Python that queries the NHTSA API to return vehicle make, model, and year. Includes error handling, history tracking, and user prompts for deletion and confirmation. Designed with efficient code logic and user experience in mind.

Cached VIN results locally to avoid redundant API calls and allow for revisiting previously decoded vehicles. This project demonstrates real-world application of APIs, JSON parsing, and conditional programming.
